## 📖 About

This project was created to provide a single platform for aptitude preparation instead of switching between multiple websites.

It focuses on the latest TCS NQT Foundation exam pattern and organizes practice into **Numerical Ability**, **Reasoning Ability**, and **Verbal Ability** with progress tracking, analytics, revision tools, and mock tests.

The dashboard is built for my own placement preparation and is also intended to help other students preparing for service-based company recruitment.

## ✨ Features

- 📊 Modern Dashboard with Progress Tracking
- 📚 Topic-wise & Subtopic-wise Practice
- ⏱️ Question & Practice Timers
- 📝 Mock Test Interface
- 📈 Performance Analytics
- 📅 Daily Study Planner
- 📖 Formula Cards
- 🔄 Revision Module
